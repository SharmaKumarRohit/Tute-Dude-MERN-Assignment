## To start the Application

    npm init

## To instal nodemon

    npm install nodemon or npm i nodemon

## To run the application

    npm run dev

## HTTP Methods

    GET         => To get/retrieve the data from the server
    POST        => Send the data to the server
    DELETE      => Deleting the data from the server
    PATCH       => Update certain fields
    PUT         => Overwriting / Full update
